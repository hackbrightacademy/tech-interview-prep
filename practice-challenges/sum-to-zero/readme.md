# Sum to Zero

Write a function that takes in a list of numbers. Return `True` if any two
 nums in the list sum to 0. Otherwise, return `False`. For this problem,
assume that there are no 0s in the list

For example:
```
>>> sum_to_zero([2, 1, 6])
False

>>> sum_to_zero([5])
False

>>> sum_to_zero([8, 2, 3, -2])
True
```
